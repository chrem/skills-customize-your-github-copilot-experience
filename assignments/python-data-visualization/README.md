# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create meaningful data visualizations using matplotlib and plotly libraries. You'll transform raw data into compelling charts and graphs that help tell stories with data.

## 📝 Tasks

### 🛠️ Create Basic Charts with Matplotlib

#### Description
Create multiple types of charts using matplotlib to visualize different datasets. You should demonstrate understanding of how different chart types are best used for different data.

#### Requirements
Completed program should:

- Create at least 3 different chart types (e.g., line plot, bar chart, scatter plot)
- Load data from a CSV file or use sample datasets
- Add appropriate titles, axis labels, and legends to each chart
- Save charts as image files (PNG or JPG)
- Display charts using `plt.show()`


### 🛠️ Customize and Style Visualizations

#### Description
Enhance your visualizations with custom styling, colors, and formatting to make them more professional and visually appealing. Experiment with different color schemes and styling options.

#### Requirements
Completed program should:

- Apply custom colors and color palettes to charts
- Adjust figure size, font sizes, and line widths
- Add grid lines and adjust transparency (alpha values)
- Create subplots to display multiple charts in one figure
- Include meaningful annotations or data labels on charts


### 🛠️ Create Interactive Visualizations with Plotly

#### Description
Use Plotly to create interactive charts that allow users to hover over data points, zoom, pan, and toggle data series on/off. Interactive visualizations enhance data exploration.

#### Requirements
Completed program should:

- Create at least 2 interactive charts using Plotly
- Include hover information that displays relevant data points
- Add interactive features like dropdown menus or buttons if applicable
- Save interactive visualizations as HTML files
- Ensure proper layout and responsive design for different screen sizes
