
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a console-based Hangman game in Python that strengthens your skills with strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Task 1: Word Selection & Game Loop

#### Description
Write the initial game structure that picks a secret word and repeatedly prompts the player for guesses until they either solve the word or run out of attempts.

#### Requirements
Completed program should:

- Use a predefined list of words.
- Randomly select one word at the start.
- Maintain and display the current progress using underscores and revealed letters.
- Count and show remaining incorrect guesses.

### 🛠️ Task 2: Guess Handling & End Conditions

#### Description
Add logic to process letter guesses, update the game state appropriately, and display the final outcome to the player.

#### Requirements
Completed program should:

- Accept single-letter guesses from the user.
- Ignore invalid input (non-letters or letters already guessed).
- Reveal correct letters in the progress display as they are found.
- End with a congratulatory message if the word is fully guessed or a failure message when no attempts remain.
